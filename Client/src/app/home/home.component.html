Eigen REST API

Zelfde thema -> Films en Directors
Beide controllers voldoen aan:
  1. Ophalen Resource Lijst
  2. Specifieke Titel/Director ophalen
  3. Resource toevoegen
  4. Resource verwijderen via ID
  5. Resource aanpassen via ID
Pagination en Sorting staan op het ophalen van een filmlijst via directorID (Gebruik 4 voor een grote lijst en pagination)
Data is beschikbaar via een DB Initializer, gebruikt gemaakt van MS SQL databank/EF Core.
Die data heeft movies gelinked met directorID.
Paginas zijn niet beschikbaar zonder gebruik te maken van Auth0 login.
